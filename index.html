About Us

The Download Inspector project is a web browser extension/application that intercepts
download requests from the browser and checks the requested files for malicious software using
an API and database. The goal of the Minimum Viable Product (MVP) is to provide basic
functionality for users to safely download files without worrying about potential malware infections.

Inspiration

This project was inspired by a problem many people face when browsing the internet.
There are often misleading links that pretend to be download links, but
they can actually be dangerous and put your personal information at risk. We addressed this
issue by creating a tool called the "Download Inspector." Instead of having to open a
separate application to check a link's safety, we made it a browser extension
for convenience. This means you can easily use it while browsing
any tab without the need for extra steps.
